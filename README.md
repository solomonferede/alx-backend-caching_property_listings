# alx-backend-caching_property_listings

A small project demonstrating caching of property listings for the ALX backend curriculum.

Prerequisites

- Python 3.9+
- Redis server

Installation

1. Clone the repo
2. python -m venv .venv && source venv/bin/activate
3. pip install -r requirements.txt

Usage

- Configure Redis connection (env vars or config file)
- Run the app: python3 app.py
- Run tests: pytest
